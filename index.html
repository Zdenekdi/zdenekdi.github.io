Jsem automation tester s 8 lety zkušeností v testování bankovních systémů a full-stack developer.

Od roku 2017 pracuji jako Automation Tester v KBC SSC Brno, kde se věnuji skriptování nových testů pro mobilní, webové a core bankovní systémy. Mám zkušenosti s testovacími frameworky jako Selenium a Robot Framework, pracuji s CI/CD pomocí Jenkins a spravuji testovací prostředí.

Nedávno jsem rozšířil své dovednosti o full-stack development, kdy jsem pracoval jako Full Stack Developer pro AT&T, kde jsem se podílel na vývoji webových aplikací, code reviews a CI/CD procesech v agilním prostředí.
